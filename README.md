## Task 3. Create Dummy Variables

Categorical variables must be converted into numerical variables before they can be used in regression analysis.

The dataset contains two categorical variables:

| Variable | Categories |
|-----------|-----------|
| `region` | East, North, South, West |
| `store_type` | Airport, High Street, Mall, Residential |

### Reference Categories

To avoid the dummy variable trap, one category from each variable was selected as the reference category.

| Variable | Reference Category |
|-----------|-------------------|
| `region` | East |
| `store_type` | Airport |

---

### Region Dummy Variables

The following dummy variables were created for `region`:

| Dummy Variable | Description |
|----------------|-------------|
| `region_North` | 1 if store belongs to North region, otherwise 0 |
| `region_South` | 1 if store belongs to South region, otherwise 0 |
| `region_West` | 1 if store belongs to West region, otherwise 0 |

Stores belonging to the East region have all region dummy variables equal to 0.

---

### Store Type Dummy Variables

The following dummy variables were created for `store_type`:

| Dummy Variable | Description |
|----------------|-------------|
| `store_type_High_Street` | 1 if store type is High Street, otherwise 0 |
| `store_type_Mall` | 1 if store type is Mall, otherwise 0 |
| `store_type_Residential` | 1 if store type is Residential, otherwise 0 |

Stores belonging to the Airport category have all store type dummy variables equal to 0.

---

### Dummy Variable Logic

For each dummy variable:

- Value = 1 indicates membership in that category.
- Value = 0 indicates non-membership in that category.
- The reference category is represented when all dummy variables for that variable are equal to 0.

---

### Avoiding the Dummy Variable Trap

Including all categories simultaneously would create perfect multicollinearity and make regression coefficients impossible to estimate correctly.

Therefore:

- `East` was excluded from the region dummy variables.
- `Airport` was excluded from the store type dummy variables.

The resulting dummy variables can now be used in regression analysis.

## Task 5: Run Multiple Regression Model

### Variables Selected

| Variable Type | Variable |
|---|---|
| Dependent Variable | `monthly_sales` |
| Numerical Predictor | `marketing_spend` |
| Numerical Predictor | `footfall` |
| Numerical Predictor | `staff_count` |
| Dummy Variable | `holiday_flag` |

### Formula Used

- `=LINEST(A8:A327,{B8:B327,C8:C327,D8:D327,E8:E327},TRUE,TRUE)`

### Formula Explanation

| Argument | Purpose |
|---|---|
| `A8:A327` | Dependent Variable (`monthly_sales`) |
| `{B8:B327,C8:C327,D8:D327,E8:E327}` | Independent Variables |
| `TRUE` | Calculate Intercept |
| `TRUE` | Return Regression Statistics |

### Regression Equation

- `monthly_sales = 387123.6029 + 1.1254(marketing_spend) + 29.3757(footfall) + 2154.8730(staff_count) + 10408.0025(holiday_flag)`

### Model Statistics

| Metric | Value |
|---|---:|
| R-Squared | 0.7848 |
| Standard Error | 48447.7123 |
| F Statistic | 287.1324 |
| Degrees of Freedom | 315 |

### P-Value Calculation

#### Step 1: Calculate t-Statistic

- `t-statistic = Coefficient / Standard Error`

#### Step 2: Calculate P-Value

- `=T.DIST.2T(ABS(t_statistic),315)`

### Business Interpretation

- `footfall` is the strongest driver of sales.
- `marketing_spend` positively impacts sales.
- `staff_count` contributes positively to sales.
- `holiday_flag` indicates higher sales during holiday periods.

### Conclusion

- Multiple regression model successfully created.
- Model explains approximately 78.48% of variation in monthly sales.
- Model is suitable for identifying key sales drivers.

#### Purpose

- Compare predicted sales with actual sales.

---

### Residual Calculation

#### Formula Used

- Residual = Actual Sales - Predicted Sales

#### Spreadsheet Formula

- `=A2-F2`

#### Interpretation

| Residual Value | Meaning |
|---|---|
| Positive | Actual sales exceeded predicted sales |
| Negative | Actual sales were lower than predicted sales |
| Near Zero | Prediction was accurate |

---

### Residual Squared

#### Formula Used

- `=G2^2`

#### Purpose

- Measures error magnitude.
- Prevents positive and negative residuals from cancelling each other.

---

### Absolute Residual

#### Formula Used

- `=ABS(G2)`

#### Purpose

- Measures the size of prediction error regardless of direction.

---
